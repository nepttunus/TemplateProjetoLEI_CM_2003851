# MVP Final

## Incluído no MVP
- Leitura do URL ativo no browser
- Botão de captura na extensão
- Envio do pedido para um motor local
- Captura de screenshot da página
- Exportação do HTML
- Recolha de metadados essenciais
- Cálculo de hashes
- Geração de manifest JSON
- Empacotamento ZIP final
- Verificação básica de integridade

## Excluído do MVP
- Multiutilizador
- Backend cloud
- Dashboard avançado
- Timestamping externo qualificado
- Assinatura digital qualificada
- Cadeia de custódia legal completa
- Recolha em lote de múltiplos alvos
